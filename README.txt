# Angelo Bruno Rizzo Photography

Sito personale fotografico di Angelo Bruno Rizzo.

## Struttura

- `index.html` = Home con video iniziale
- `portfolio.html` = Galleria fotografica
- `about.html` = Biografia
- `contatti.html` = Contatti
- `style.css` = Stile grafico del sito
- `script.js` = Script JavaScript
- `CNAME` = collegamento dominio personalizzato

## Cartelle

- `images/` = inserire qui le fotografie
- `video/` = inserire qui il video della home

## File da inserire

Nella cartella `video/` inserire il file:

`home.mp4`

Nella cartella `images/` inserire le fotografie con questi nomi iniziali:

- foto1.jpg
- foto2.jpg
- foto3.jpg
- foto4.jpg
- foto5.jpg
- foto6.jpg
- foto7.jpg
- foto8.jpg
- foto9.jpg
- about.jpg

## Dominio

Il file `CNAME` contiene:

www.angelobrunorizzo.com

Questo permette di collegare GitHub Pages al dominio personalizzato.
